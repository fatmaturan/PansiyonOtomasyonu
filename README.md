# 🏨 Pansiyon Otomasyonu

C# Windows Forms kullanılarak geliştirilen masaüstü pansiyon otomasyon sistemi. Uygulama; müşteri kayıtlarının yönetimi, oda takibi, gelir-gider işlemleri, hava durumu ve radyo gibi ek özellikleri tek bir arayüz üzerinden sunmaktadır.

## 🚀 Özellikler

- 👤 Müşteri kayıt işlemleri
- 🛏️ Oda doluluk ve boş oda takibi
- 💰 Gelir - Gider yönetimi
- 📰 Güncel gazete bağlantıları
- 🌦️ OpenWeatherMap API ile hava durumu bilgisi
- 📻 İnternet üzerinden radyo dinleme
- 💬 Yorum ve not ekranı
- ℹ️ Hakkımızda sayfası
- 📊 Yönetim paneli

## 🛠️ Kullanılan Teknolojiler

- C#
- Windows Forms (.NET Framework)
- SQL Server
- OpenWeatherMap API
- Visual Studio

## 📂 Proje Yapısı

```
PansiyonOtomasyonu
│
├── Formlar
│   ├── Müşteriler
│   ├── Odalar
│   ├── Gelir-Gider
│   ├── Gazeteler
│   ├── Hava Durumu
│   ├── Radyo
│   └── Yönetim Paneli
│
├── SQL Veritabanı
└── Kaynak Dosyaları
```

## 🌦️ Hava Durumu

Hava durumu bilgileri **OpenWeatherMap API** kullanılarak internet üzerinden anlık olarak alınmaktadır.

## 📻 Radyo

Radyo ekranında internet üzerinden yayın yapan radyo kanallarının canlı yayın bağlantıları kullanılarak müzik dinlenebilmektedir.

## 💾 Veritabanı

Uygulama SQL Server veritabanı kullanmaktadır.

Temel tablolar:

- Müşteriler
- Odalar
- Gelir-Gider
- Yönetici
